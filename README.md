IA04 Backend (Express + MongoDB)

Function description
- JWT authentication API issuing access and refresh tokens.
- Endpoints: POST /auth/register, POST /auth/login, POST /auth/refresh, GET /auth/me (protected), POST /auth/logout.
- Access token verifies protected routes; refresh token issues new access tokens.
